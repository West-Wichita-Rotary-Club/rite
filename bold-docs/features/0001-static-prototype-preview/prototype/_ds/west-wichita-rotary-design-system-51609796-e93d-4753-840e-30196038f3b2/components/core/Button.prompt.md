Primary call-to-action control — use for "Register", "Donate", "RSVP", "Learn more" across the club site.

```jsx
<Button variant="primary">Register for event</Button>
<Button variant="gold">Donate</Button>
<Button variant="outline">Learn more</Button>
```

Variants: `primary` (Azure fill, main actions), `gold` (Rotary Gold fill, donate/celebratory), `outline` (Royal Blue border, secondary), `ghost` (text-only, tertiary/nav-adjacent). Sizes: `sm`, `md`. Pass `href` to render as a link instead of a button.
