Content container for meeting info, service project summaries, and membership blurbs — the gold top band is the club's signature card treatment (from the original starter kit).

```jsx
<Card title="Weekly Meeting">
  Thursdays at noon — join us for fellowship, service updates, and guest speakers.
</Card>
```

`accent="azure"` for a secondary-emphasis card; `accent="none"` or `bandless` for a flat card with no band.
