# RITE Backbone

Product Owner TL;DR: This repo builds the Rotary Inter-country Teacher Exchange community archive: a bilingual platform for preserving RITE history, connecting alumni, and supporting future participation across Wichita, Panama, and Argentina.

## Principles

1. **Product identity is fixed.** RITE means Rotary Inter-country Teacher Exchange, a West Wichita Rotary / Rotary District 5680 program preserving the teacher exchange begun in 1996. `status: enforced` `source: ratified(discovery-playback, 2026-07-15)`
2. **The product is a living archive and community platform.** Core scope includes the history timeline, story archive, alumni directory, school and club profiles, community registration, resource sharing, and future participant workflows. `status: enforced` `source: provided(design-spec, 2026-07-15)`
3. **Privacy review gates publication and private data.** Editors or administrators review submissions for historical accuracy, child privacy, school privacy, and member safety before public release or directory access. `status: enforced` `source: ratified(discovery-playback, 2026-07-15)`
4. **English and Spanish are first-class.** User-facing content, navigation, core workflows, and stored content model decisions must support bilingual English/Spanish operation from the start. `status: enforced` `source: ratified(discovery-playback, 2026-07-15)`
5. **Primary personas are explicit.** Teachers, host families, Rotary leaders, coordinators, and school administrators are first-class users; anonymous visitors can read approved public archive content. `status: enforced` `source: provided(design-spec, 2026-07-15)`
6. **The prototype is product evidence.** `bold-docs/features/0001-static-prototype-preview/prototype/` is the archived feature prototype and should be used as reference material for product intent, flows, content, and visual direction until replaced by production implementation; it is not final site source. `status: enforced` `source: ratified(discovery-playback, 2026-07-15)`
7. **Use the West Wichita Rotary design system.** UI should follow the discovered Rotary brand foundations: Azure, Royal Blue, Sky Blue, Rotary Gold, Cool/Warm Gray neutrals, Oswald/Open Sans substitutes unless licensed fonts are supplied, small radii, and gold/azure accent bands. `status: enforced` `source: discovered(bold-docs/features/0001-static-prototype-preview/prototype/_ds)`
8. **Use system primitives and tokens in production UI.** Buttons, cards, badges, callouts, and area-of-focus tags should follow the prototype design-system contracts; new production code should avoid raw colors, raw spacing, and off-system fonts. Legacy prototype inline styles are grandfathered. `status: adopting` `source: discovered(bold-docs/features/0001-static-prototype-preview/prototype/_ds/*_adherence.oxlintrc.json)`
9. **Production architecture is .NET API plus standalone React.** Build an ASP.NET Core MVC/Web API backend backed by SQLite and a standalone React site that calls the API. `status: enforced` `source: provided(stack-answer, 2026-07-15)`
10. **Azure is the hosting and media direction.** Plan for Azure hosting and Azure Blob Storage for high-resolution photos, videos, transcripts, and other archive media. `status: adopting` `source: provided(design-spec, 2026-07-15)`
11. **Testing and CI are not yet established.** Future work must make explicit choices for test coverage, build validation, and deployment checks before treating the app as production-ready. `status: adopting` `source: discovered(repo-inventory, 2026-07-15)`
